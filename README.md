# 🎮 GameHub Pro

GameHub Pro is a full-stack game library web application built with Spring Boot, Thymeleaf, and MySQL.

## 🚀 Features
- Browse games
- View game details
- Trailer popup modal
- Related games section
- Clean layered architecture

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- HTML, CSS, JavaScript

## 📂 Project Structure
src/main/java/com/dulith/gamehub
- controller
- service
- repository
- entity
- config

src/main/resources
- templates
- static
- application.properties

## ▶️ How to Run
1. Create a MySQL database named `gamehub_db`
2. Update `application.properties` with your MySQL username and password
3. Run the project with:

```bash
mvnw.cmd spring-boot:run
